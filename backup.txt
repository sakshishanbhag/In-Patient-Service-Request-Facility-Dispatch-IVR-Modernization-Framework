'
# IVR Hospital Service Request System

## Overview

This project implements an **Interactive Voice Response (IVR)** system for hospital inpatient service requests using **Twilio** and **FastAPI**. Patients can call a Twilio number and use **speech recognition** or **DTMF (keypad input)** to request services like:

- **Housekeeping** (cleaning, spill cleanup, fresh linens)
- **Maintenance**
- **IT Support**
- **Nursing Station**

### Key Features
- Natural language speech input (English - India)
- Fallback DTMF input (press 1-4)
- Polly.Aditi neural voice for natural Indian-accented responses
- In-memory session tracking for multi-turn conversations
- Local development with ngrok tunneling
- Automated testing with FastAPI TestClient
- One-click call triggering for testing

## Architecture

```
User Call → Twilio → ngrok → FastAPI (/voice) → TwiML Response
                           ↳ Speech/DTMF → Intent Handling → Service Response
```

- **backend.py**: Core FastAPI app with TwiML endpoints (`/voice`, `/handle-intent`, `/handle-housekeeping`)
- **trigger_call.py**: Script to initiate test calls via Twilio API
- **test_ivr.py**: Unit tests for voice endpoints
- **ngrok.exe**: For local webhook exposure (Twilio → localhost)

## Setup

### 1. Clone & Install Dependencies
```i
pip install fastapi uvicorn twilio python-dotenv pytest
```

### 2. Environment Configuration
Copy `.env.example` to `.env` and fill in your credentials:
```
TWILIO_ACCOUNT_SID = <your_account_sid>
TWILIO_AUTH_TOKEN = <your_auth_token>
TWILIO_PHONE_NUMBER = <your_twilio_number>
USER_PHONE = <your_phone_number>
NGROK_URL=https://abc123.ngrok.io  
```


### 3. Expose Local Server with ngrok
```bash
# Terminal 1: Start FastAPI server
uvicorn backend:app --host 0.0.0.0 --port 8000 --reload

# Terminal 2: Start ngrok (updates NGROK_URL automatically)
./ngrok.exe http 8000
```
Copy the ngrok HTTPS URL (e.g., `https://abc123.ngrok.io`) to `.env`.

### 4. Configure Twilio Webhook
In Twilio Console → Phone Numbers → Your Number → Webhooks:
- **Voice**: `https://your-ngrok-url.ngrok.io/voice` (HTTP POST)

## Usage

### Test a Call
```bash
python trigger_call.py
```
This dials `USER_PHONE` and starts the IVR flow.

### Interactive Flow
```
Welcome... say "housekeeping", "maintenance", etc.

Housekeeping → "cleaning", "spill", "fresh linens"
Other intents → Auto-hangup with confirmation
Invalid → Redirect to main menu
```

### Run Tests
```bash
pytest test_ivr.py
```

## Endpoints

| Endpoint          | Method | Description                  |
|-------------------|--------|------------------------------|
| `/voice`          | POST   | Main IVR menu                |
| `/handle-intent`  | POST   | Process service selection    |
| `/handle-housekeeping`|POST| Housekeeping sub-options     |

## Development

- **Voice**: Customize `IVR_VOICE = "Polly.Aditi"` (other options: `manoj`, `woman`, etc.)
- **Sessions**: `call_sessions` dict (in-memory; use Redis for production)
- **Expand**: Add more intents, database logging, SMS fallbacks

## Production Deployment

1. Deploy FastAPI to Render/Heroku/Railway
2. Use Twilio Production URL (no ngrok)
3. Redis for sessions
4. Database for request logging
5. Multiple languages/voices

## Troubleshooting

- **No speech recognition**: Check Twilio → Voice → Speech settings
- **Webhook errors**: Verify ngrok URL in `.env` and Twilio console
- **Call fails**: Check Twilio credits & phone number status

import os
from dotenv import load_dotenv
from fastapi import FastAPI, Form
from fastapi.responses import Response
from twilio.twiml.voice_response import VoiceResponse, Gather

load_dotenv()

app = FastAPI()

# Load ngrok URL from .env
BASE_URL = os.getenv("NGROK_URL")

# Voice configuration
IVR_VOICE = "Polly.Aditi"

# Session storage
call_sessions = {}


@app.post("/voice")
async def main_menu():

    response = VoiceResponse()

    gather = Gather(
        input="speech dtmf",
        num_digits=1,
        action=f"{BASE_URL}/handle-intent",
        method="POST",
        language="en-IN",
        speechTimeout="auto"
    )

    gather.say(
        "Welcome to the In Patient Service Request system. "
        "How can I help you today? "
        "You may say housekeeping, maintenance, I T support, or nursing station.",
        voice=IVR_VOICE
    )

    response.append(gather)
    response.redirect(f"{BASE_URL}/voice")

    return Response(content=str(response), media_type="application/xml")


@app.post("/handle-intent")
async def handle_intent(
    CallSid: str = Form(...),
    SpeechResult: str = Form(default=None),
    Digits: str = Form(default=None)
):

    response = VoiceResponse()

    text = ""
    if SpeechResult:
        text = SpeechResult.lower()

    if CallSid not in call_sessions:
        call_sessions[CallSid] = {}

    if Digits == "1" or "housekeeping" in text or "clean" in text:

        call_sessions[CallSid]["intent"] = "housekeeping"

        gather = Gather(
            input="speech dtmf",
            num_digits=1,
            action=f"{BASE_URL}/handle-housekeeping",
            method="POST",
            language="en-IN",
            speechTimeout="auto"
        )

        gather.say(
            "What housekeeping service is needed? "
            "Say cleaning, spill, or fresh linens.",
            voice=IVR_VOICE
        )

        response.append(gather)

    elif Digits == "2" or "maintenance" in text:

        response.say(
            "Maintenance request registered. "
            "A technician will arrive shortly.",
            voice=IVR_VOICE
        )
        response.hangup()

    elif Digits == "3" or "it support" in text:

        response.say(
            "I T support request registered.",
            voice=IVR_VOICE
        )
        response.hangup()

    elif Digits == "4" or "nurse" in text:

        response.say(
            "Connecting to nursing station.",
            voice=IVR_VOICE
        )
        response.hangup()

    else:

        response.say(
            "Sorry, the request was not understood.",
            voice=IVR_VOICE
        )
        response.redirect(f"{BASE_URL}/voice")

    return Response(content=str(response), media_type="application/xml")


@app.post("/handle-housekeeping")
async def handle_housekeeping(
    CallSid: str = Form(...),
    SpeechResult: str = Form(default=None),
    Digits: str = Form(default=None)
):

    response = VoiceResponse()

    text = ""
    if SpeechResult:
        text = SpeechResult.lower()

    if Digits == "1" or "clean" in text:

        response.say(
            "Routine cleaning request registered. "
            "Housekeeping staff will arrive shortly.",
            voice=IVR_VOICE
        )

    elif Digits == "2" or "spill" in text:

        response.say(
            "Urgent spill cleanup reported. "
            "An emergency janitorial team has been dispatched.",
            voice=IVR_VOICE
        )

    elif Digits == "3" or "linen" in text:

        response.say(
            "Fresh linens will be delivered shortly.",
            voice=IVR_VOICE
        )

    else:

        response.say(
            "Request not understood.",
            voice=IVR_VOICE
        )

    response.hangup()

    return Response(content=str(response), media_type="application/xml")
